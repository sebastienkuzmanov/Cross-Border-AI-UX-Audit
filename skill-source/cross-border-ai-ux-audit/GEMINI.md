# GEMINI.md

This project is an **Agent Skill**: **Cross-Border AI UX Audit** (跨境 AI 用户体验审计).

Use **`SKILL.md`** as the authoritative instructions when the user asks to check whether an AI product's UX is ready for a target market, to run a localization / market-readiness / market-fit review, or to audit a chatbot/agent/app for a specific country or language market. Load `references/` on demand and use the Python helpers in `scripts/` for scoring, rendering, and exports.

For installation across agents (including Gemini CLI's `~/.gemini/skills/`), see `INSTALL.md`. Output reports bilingually (English + 中文). Escalate legal/compliance/security/accessibility-conformance items to qualified experts rather than giving a verdict.
